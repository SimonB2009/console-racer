# console-racer
Das ist meine erste kleine **Konsolen Applikation** mit der ich (mit 13 Jahren **:-)** )C# lernen will

# Spielanleitung
Das Ziel ist es das Auto (!) in der Strecke zu halten. Das Auto kannst du mit **a** (rechts) und **d** (links) bewegen.
Dein Highscore wird in einer Tabelle gespeichert,sodass du beim nächsten mal vieleicht ihren Highscore verbessern können.
Die Tabelle kannst du im Hauptmenü mit der Taste **h** öffnen.
